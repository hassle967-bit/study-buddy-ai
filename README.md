# Study Buddy AI

## Project Description
Study Buddy AI is an AI-powered learning assistant designed to help students with their studies. It can provide explanations, answer questions, and support students in understanding different topics.

## Features
- AI-based question answering
- Study assistance
- Easy-to-use interface
- Helps students learn faster

## Technologies Used
- HTML
- CSS
- JavaScript
- AI API / AI Tools

## How to Run the Project
1. Open the project files.
2. Install required dependencies.
3. Run the application.
4. Start using Study Buddy AI.
